# 🎬 Netflix Exploratory Data Analysis

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset using Python. It explores trends in content type, countries, ratings, genres, directors, and release years.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab

## Analysis Performed
- Data Cleaning
- Missing Value Analysis
- Duplicate Removal
- Movies vs TV Shows
- Top Countries
- Ratings Distribution
- Content Released by Year
- Top Directors
- Genre Analysis

## Key Insights
- Movies are more common than TV Shows.
- The United States has the highest number of Netflix titles.
- TV-MA is among the most common content ratings.
- Netflix's content library has grown significantly over the years.
